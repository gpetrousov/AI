Greedy hill climbing algorithm written in python2 by Giannis Petrousov. The program places queens and knights in an 8X8 
chessboard so they dont collide with each other. This was an essay for the class Artificial Intelligence.

lazy_bot.py
This runs all the functions in sequential order.

pieces_class.py
Contains the class for the queens and knights.

restarts.py
Greedy hill climbing with restarts.

sideways.py
Greedy hill climbinb with sideways moves and restarts.

stochastic.py
Greedy hill climbing with stochastic choices and restarts.
